# git, github
## 1. git이란?
```
버전관리 소프트웨어이다(컴퓨터 말고 다른곳에 저장하는 거)
로컬 저장소에서 작동하기에 다른 사람과 작업을 공유하고 협업하기 어렵다.
로컬 저장소의 모든 변경사항을 기록하여 파일의 버전관리가 용이해진다.
인터넷 없이 이용할 수 있다
```
## 2. github란?
```
git을 사용하는 프로젝트를 지원하는 웹 호스팅 서비스(git을 관리 할 수 있는 곳)
github는 웹 상에서 클라우드 서버를 통해 로컬 저장소의 코드를 업로드하고 공유할 수 있다(다른 사람의 프로젝트를 가져와서 수정 가능)
 Github에서 Code Review 등을 통해 협업이 가능하고, 수많은 오픈 소스 프로젝들이 Github로부터 호스팅되고 있어서, 누구든 자유롭게 기여할 수 있다.
 백업본을 저장해 둔다면 내 파일들을 복구시킬 수 있다.
```
