# HTML 기본 명령어
### 문서의 기본 구조를 나타내 주는 태그
- HTML 언어로 작성되어 있다는 것을 알려주는 태그
```
	<HTML>...</HTML>
```
- Heading의 준말로 글의 머리말에 해당하는 태그
```
	<HEAD>...</HEAD>
```
- 본문에 해당하는 부분을 알려주는 태그
```
	<BODY>...</BODY>
```
- 타이틀바에 새겨질 글자를 정의하는 태그
```
	<TITLE>...</TITLE>
```
- 표제 부분에 들어갈 말을 정의 마크다운 #와 반대인 태그(1~6)
```
	<Hn>...</Hn>
```
- 주석을 달거나 잠시동안 이미지를 보이지 않게 할 때 사용하는 태그
```
	<!-주석내용->
```
### 문단 혹은 줄을 바꾸는 태그
- 문단을 바꾸는 태그, 줄바꿈과 동시에 줄을 뛴 것같은 효과가 나타나는 태그
```
	<P>
```
- 문단을 바꾸는 태그,줄바꿈의 역할만을 수행
```
	<BR>
```
- 여백이나 줄간격 등을 고정시켜 주는 역할을 합니다.
```
	<PRE>...</PRE>
```
### 글자의 크기를 마음대로 조절하는 태그
- 글자의 크기를 마음대로 조절할 수 있는 태그, 마크다운 #과 비슷한 태그(1~7)
```
	<FONT SIZE=n>...</FONT>
```
### 선그리기 태그
- 입체적인 선을 그려주는 태그. ALIGN은 선의 정렬, WIDTH는 선의 폭, SIZE는 선의 높이
```
	<HR ALIGN= WIDTH= SIZE=>
```
### 이미지 맵을 처리하는 태그
- 이미지맵을 정의하는 태그
```
	<IMG SRC="..." ISMAP>
```
- 넷스케이프2.0에서 이미지맵을 정의하는 태그
```
	<MAP NAME=...>...</MAP>
```
### 목록을 정리해 주는 태그
- 순서가 없는 목록으로 일반적인 나열을 하는 태그
```
	<UL>...</UL>
```
- 순서가 있는 목록으로 위에서부터 번호를 매기는 태그
```
	<OL>...</OL>
```
- 메뉴 목록으로 그리길지 않은 문장의 열거에 사용하는 태그
```
	<MEUN>...</MEUN>
```
- 디렉토리 목록으로 메뉴 목록보다 짧은 문장을 나열하는 태그
```
	<DIR>...</DIR>
```
- 정의 목록 태그
```
	<DL>...</DL>
```
### 문서를 가운데로 정렬시켜 주는 태그
- 전체 문장을 가운데로 정렬시켜 주는 태그
```
	<CENTER>...</CENTER>
```
 ### 인용하기 태그
 - 문장 내에서 인용을 할 경우 들여쓰기를 한 후 인용구로 처리하는 태그
 ```
 	<BLOCKQUOTE>...</BLOCKQUOTE>
 ```
 ### 글자의 모양을 정의해 주는 태그
 - 굵은 글씨를 나타내 주는 태그
 ```
 	<STRONG>...</STRONG>
 ```
 ```
 	<B>...</B>
 ```
 - 이탤릭체의 글씨를 나타내 주는 태그
 ```
 	<EM>...</EM>
 ```
 ```
 	<I>...</I>
 ```
 - 타자기체의 글씨를 나타내 주는 태그
 ```
 	<KBD>...</KBD>
 ```
 ```
 	<CODE>...</CODE>
 ```
 ```
 	<TT>...</TT>
 ```
 ### 배경 이미지 작업하기
 - 배경 이미지를 띄워주는 태그
 ```
 	<BODY BACKGROUND="...">
 ```
 - 배경 색깔을 지정해 주는 태그
 ```
 	<BODY BGCOLOR="#nnnnnn">
 ```
 ### 그밖의 태그들
 - 정의되어지는 단어에 대한 것을 말하는 태그
 ```
 	<DFN>...</DFN>
 ```
 - 책이나 사진 등의 제목을 말할 때 스는 태그
 ```
 	<CITE>...</CITE>
 ```
 - 컴퓨터상에 메시지를 나타낼 때 쓰는 태그
 ```
 	<SAMP>...</SAMP>
 ```
 - 이텔릭체로 표현하는 태그
 ```
 	<VAR>...</VAR>
 ```
 - 점자에 관한 태그
 ```
 	<SUB>...</SUB>
 ```
 - 윗첨자를 나타내는 태그
 ```
 	<SUP>...</SUP>
 ```
 - 기본적으로 미리 약속된 크기를 다시 정할 때 사용하는 태그
 ```
 	<BASEPOINT>...<BASEPOINT>
 ```
 ### 부분을 강조해 주는 태그
 - 글자를 깜박이게 해주는 태그
 ```
 	<BLNK>...</BLNK>
 ```
 - 글자에 밑줄을 그어주는 태그
 ```
 	<U>...</U>
 ```
 ### 연결하기 태그
 - 다른 홈페이지와 연결시켜주는 태그
 ```
 	<A HREF="...">...</A>
 ```
 - 자신의 홈페이지 내에서 연결해주는 태그
 ```
 	<A NAME="...">...</A>
 ```
 ### 주소 및 편지 서비스
 - 주소에 대한 정의를 내려주는 태그
 ```
 	<ADDRESS>...</ADDRESS>
 ```
 - 편지쓰기 창을 띄워주는 태그
 ```
	<A HREF="MAILTO...">
 ```
 ### 색상 지정하기
 - 글자색을 지정하는 태그
 ```	
 	<BODY TEXT= "#nnnnnn">...</BODY>
 ```
 - 하이퍼링크의 색을 지정하는 태그
 ```
 	<BODY LINK= "#nnnnnn">...</BODY>
 ```
 - 한 번 누른적이 있는 하이퍼링크의 색을 지정하는 태그
 ```
 	<BODY VLINK= "#nnnnnn">...</BODY>
 ```
 - 누르고 있는 동안 색을 지정하는 태그
 ```
 	<BODY ALINK= "#nnnnnn">...</BODY>
 ```
 ### 표 만들기 태그
 - 표의 전체 형식을 나타내는 태그
 ```
 	<TABLE BORDER>... </TABLE>
 ```
 - 표 내부에 들어갈 형식을 정의하는 태그
 ```
 	<TR><TD></TD></TR>
 ```
 - 가로칸을 n만큼 합쳐주는 태그
 ```
 	<TD COLSPAN=n>
 ```
 - 세로칸을 n만큼 합쳐주는 태그
 ```
 	<TD ROWSPAN=n>
 ```
 - 표안의 좌우 정렬 방식을 정의하는 태그
 ```
 	<TD ALIGN=...>
 ```
 - 표안의 상하 정렬 방식을 정의하는 태그
 ```
 	<TD VALIGN=...>
 ```
 ### 문서를 임의로 고정시키는 태그
 - 화면 크기에 따라 문단이 바뀌는 것을 방지하는 태그
 ```
 	<NOBR>...</NOBR>
 ```
 - 위의 태그 안에서 문단을 바꿀 때 사용하는 태그
 ```
	<WBR>...</WBR>
 ```
### CGI 양식 첨가하기
- 양식을 정의하는 태그
```
	<FORM>...</FORM>
```
- 양식의 방법을 정의하는 태그
```
	<FORM METHOD="..." ACTION="...">
```
- 글틀 상자를 정의하는 태그
```
	<TEXTAREA NAME="..." ROWS="...">
```
- 어떤 형태로 입력할 것인지를 정의합니다.
```
	<INPUT TYPE="...">
```
- 라디오 단추를 만들어주는 태그
```
	<INPUT TYPE="RADIO">
```
- 체크 상자를 만들어주는 태그
```
	<INPUT TYPE= "CHECKBOX">
```
- 제출 버튼을 만들어 주는 태그
```
	<INPUT TYPE= "SUBMIT">
```
- 취소 버튼을 만들어 주는 태그
```
	<INPUT TYPE="RESET">
```
- 선택 사항 상자를 만들어주는 태그
```
	<SELECT NAME="...">
```
### 창만들기 태그
- 창의 기본 틀을 지정하는 태그
```	
	<FRAMESET>... </FRAMESET>
```
- 위 아래로 나누는 태그
```
	<FRAMESET ROW= "숫자 혹은 %">
```
- 좌우로 나누는 태그
```
	<FRAMESET COLS= "숫자 혹은 %">
```
- 창에 들어갈 좌우 여백을 정의하는 태그
```
	<FRAME SRC="..." MARGINWIDTH="...">
```
- 창에 들어갈 상하 여백을 정의하는 태그
```
	<FRAME SRC="..." MARGINHEIGHT="...">
```
- 스크롤바를 설정해주는 태그
```
	<FRAME SCROLLING= "...>
```
- 프레임이 안보이게 해주는 태그
```
	<NOFRAMES>... </NOFRAMES>
```
### 이미지 다루기에 필요한 태그
- 이미지를 정의해주는 태그
```
	<IMG SRC="...">
```
- 이미지의 정렬 상태를 나타내주는 태그
```
	<IMG SRC="..." ALIGN=...>
```
- 이미지의 크기를 조절해주는 태그
```
	<IMG SRC="..." WIDTH="..." HEIGHT="...">
```
- 이미지를 JPEG파일로 나타내 주는 태그
```
	<IMG SRC="..." LOWSRC="...">
```
- 이미지의 여백을 조절해주는 태그
```
	<IMG SRC="..." VSPACE= HSPACE=>
```
- 이미지의 테두리선을 조절하는 태그
```
	<IMG SRC="..." BORDER="...">
```
- 이미지가 전송되지 않았을 경우에 글자로 대처해주는 태그
```
	<IMG SRC="..." ALT="...">
```
