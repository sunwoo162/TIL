### type선택자
> 
* 선택자중 가장 간단한 선택자
* html문서의 태그 이름을 선택자로 사용할 수 있다.
```
ex) p{
		color: red;
     }
     <p>안녕하세요</p>
```
---
### id선택자
* html 문서의 요소 중 같은 요소라도 각기 다른 이름을 지정하여 따로 속성을 부여할 수 있다.
* html 요소에 id로 이름을 붙일 때에는 유일한 이름을 부여해야하고 다른 요소에 같은 id를 주어서는 안된다.
* css에서는 id 선택자 앞에'#'를 붙여야 한다.
* id명은 숫자로 시작할 수 없고 주로 영문자로 시작한다.
```
ex) #red{
		color: red;
     }
     <p id="red">이것은 id선택자입니다. </p>
     <p>이것은 id선택자입니다.</p>
```
### class선택자
* html 문서의 여러 요소 중 같은 이름을 갖는 요소만 모아 따로 속성을 부여할 수 있다.
* html 요소에 같은 이름을 부여할 때에는 class로 이름을 붙인다.
* 여러 개의 요소에 같은 class명을 부여할 수 있고 하나의 요소에 여러 개의 class명을 부여할 수도 있다.
* css에서는 class 선택자 앞에'.'을 붙여야한다.
```
ex) .red{
		color: red;
     }
     .blue {
     	 font-weight: bold;
     }
```
### 전체 선택자
* 페이지의 모든 요소를 가리키는 선택자로 '*'로 표시한다
* id나 class명 부여와 상관없이 속성을 부여할 수 있다.
```
	*{
    	color: red;
     }
     #
```
### 하위 선택자
* 요소 내부에 있는 모든 해당 요소를 가리킨다.
* css에서는 선택자 사이를 공백으로 분리한다.
```
.red p {
      color: red;
 }
 <div class="red">
  <p>안녕하세요</p>
  <u1>
       <li><p>이것은 하위 선택자입니다.</p></li>
       <li>이것은 하위 선택자입니다.</li>
  </ul>
</div>
<p>이것은 하위 선택자입니다.</p>
```
### 자식 선택자
* 요소 내부에 있는 해당 요소를 가리키지만 하위요소의 하위요소는 가리키지 않는다.
* css에서는 선택자 사이를 > 로 분리한다.
```
.red > p{
    color: red;
 }
<p>이것은 자식 선택자입니다.</p>
<div class="red">
    <p>이것은 자식 선택자입니다.<p>
    <ul>
        <li><p>이것은 자식 선택자입니다.</p></li>
    </ul>
</div>
```
### 인접 선택자
* 현재 요소의 바로 뒤에 나오는 요소만을 가리키는 선택자이다.
*  css에서는 선택자 사이를 '+'로 분리한다.
h2 + p{
     color : red; 
 }
 <h2>이것은 인접 선택자입니다.</h2>
 <p> 이것은 인접 선택자 입니다.</p>
 <p>이것은 인접 선택자 입니다.</P>
  <h2>이것은 인접 선택자입니다.</h2>
   <p>이것은 인접 선택자 입니다.</P>
    <p>이것은 인접 선택자 입니다.</P>

### 형제 선택자
* 현재 요소와 같은 계층에 있는 요소만을 선택할 수 있다.
* css에서는 선택자 사이를 '~'로 분리한다.
```
h2 ~ p{
    color: red;
}
 <h2>이것은 형제 선택자입니다.</h2>
 <p>이것은 형제 선택자 입니다.</p>
 <u1>
    <li><p>이것은 형제 선택자입니다.</p></li>
</ul>
<p>이것은 형제 선택자 입니다.</p>
 ```
### 그룹 선택자
* 여러 선택자들을 함께 묶어 속성을 부여한다.
* css에서는 선택자 사이를 ','로 분리한다.
```
h2,p{
    color: red;
}
<h1> 이것은 그룹 선택자입니다.</h1>
<h2>이것은 그룹 선택자입니다.</h2>
<p>이것은 그룹 선택자입니다.</p>
<u1>
    <li><p>이것은 그룹 선택자입니다.</p></li>
</ul>
```
### 속성 선택자
* html 요소의 속성 유무 또는 속성값을 중괄호 안에 넣어 선택자로 사용할 수 있다.
```
p[class |= "red"]{
    color: red;
}
p[class $="blue"]{
     color: blue
}
<p class="red">이것은 속성 선택자 입니다.</p>
<p class="redblue">이것은 속성 선택자 입니다</p>
<p class="red-1">이것은 속성 선택자 입니다</p>
<p class="blue">이것은 속성 선택자 입니다.</p>
```
### 가상 클래스 선택자
*  링크가 걸린 무낮에 스타일을 부여하는 것이다.
```
a:focus{
color: yellogreen;
}
<u1>
    <li><a href="#">abc</a></li>
    <li><a href="#">aaa</a></li>
    <li><a href="#">bbb</a></li>
    <li><a href="#">ccc</a></li>
```
### 가상 요소 선택자
* 요소에 id이나 class명을 부여하지 않고도 위치를 찾아서 선택할 수 있는 선택자이다.
```
p:first-child{
    color: red;
}
p:nth-child(3){
    color:blue;
}
p:before{
    content:"!";
}
p:after{
    content:"$";
}
<div>
    <p>abc</p>
    <p>aaa</p>
    <p>bbb</p>
    <p>ccc</p>
</div>
```
### 종속 선택자
* type 선택자와 id 선택자, class 선택자가 결합된 형태이다
<p id="abc">종속 선택자</p>

### 선택자의 우선순위
* 같은 선택자가 여러 css 명령을 중복을 받으면 우선 순위가 같은 선택자는 나중에 기술한 것이 먼저 기술한 것보다 우선순위가 높다.
* 최우선으로 css를 적용해야 할 경우에는 선택자 속성을 부여하는 마지막에 "!important"를 붙이면 된다.
* 전체 우선 순위
```
Type선택자 < class선택자,가상선택자 < id선택자 < inline style < !import
```
