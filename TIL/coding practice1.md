# 코드
- html로 버튼을 만들고 그 안에 "눌러"라는 말을 넣어 눌러를 css로 색깔과 크기 위치를 바꾼 코드이다
### HTML code
```
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="ll.css">  <!-- css.css와 연결! -->
</head>
<body>
    <button class="my-class">눌러</button>
</body>
</html>
```
### CSS code
```
.my-class{
    color: blue;
    font-size : 40px;
    text-align: center;
}
```
